# List of Turtle Functions

**Moving**
- up()
- down()
- left(*angle*), right(*angle*)
- forward(*angle*)
- setheading(*angle*)
- circle(*radius*)

**Checking State**
- isdown()
- heading()
- xcor(), ycor()

**Color**
- pensize(*size*)
- color(*color*)
- fillcolor(*color*)
- begin_fill()
- end_fill()

**Extra**
- screensize(*width*, *height*)
- done(): keep the window open
- speed()
